Webamp — the hologram-native Winamp engine (a pixel-exact, open reimplementation of
Winamp 2.9 that runs the ORIGINAL Winamp classic-skin spec in the browser).

Pinned, integrity-verified external authority (Law L5) — vendored dist build + the
canonical classic base skin, NOT loaded from a CDN. winamp.html loads exactly these
bytes; winamp-witness.mjs re-derives their sha256 and refuses a drift. The bundle is
self-contained (React + the audio engine + the default classic skin embedded) — one
engine file, no extra import — so the whole player (main window · equalizer · playlist ·
the spectrum-analyzer / oscilloscope visualizers driven by the Web Audio AnalyserNode)
works offline in any browser.

Why this IS the original Winamp spec: Webamp is reverse-engineered to be pixel-faithful
to Winamp 2.9, and it consumes the unmodified original skin format — a `.wsz` is just a
renamed ZIP of the original sprite bitmaps (MAIN.BMP, CBUTTONS.BMP, TITLEBAR.BMP,
NUMBERS.BMP, TEXT.BMP, VOLUME.BMP, BALANCE.BMP, POSBAR.BMP, PLAYPAUS.BMP, EQMAIN.BMP,
PLEDIT.BMP) plus the text configs (PLEDIT.TXT, VISCOLOR.TXT, REGION.TXT). The vendored
base-2.91.wsz is that exact archive; drag any real Winamp skin onto the player and it
applies, because the player implements the spec, not a lookalike.

Milkdrop (the AVS/butterchurn preset visualizer) is an OPTIONAL lazy import and is NOT
vendored — so Milkdrop is unavailable offline by design; the built-in classic main-window
visualizers (spectrum analyzer + oscilloscope) are part of the bundle and always present.
