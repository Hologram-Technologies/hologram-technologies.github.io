video.js — the native video player component (HTML5 <video> + MSE adaptive streaming).

Pinned, integrity-verified external authority (Law L5) — vendored dist builds, not
loaded from a CDN. video.html loads exactly these bytes; video-witness.mjs re-derives
their sha256 and refuses a drift. The HLS/DASH source handler (@videojs/http-streaming,
"VHS") is bundled INTO video.min.js — one engine file, no extra import — so Media Source
Extensions adaptive streaming (up to 4K/8K, ABR + Low-Latency HLS) works offline. The
icon font is embedded as a base64 woff data-URI inside the CSS, so there is NO external
font file: the player is exactly two content-addressed bytes-objects + a license.
