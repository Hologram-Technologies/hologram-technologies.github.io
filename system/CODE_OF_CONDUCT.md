# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1.

We pledge to make participation a harassment-free experience for everyone. We act and
interact in ways that contribute to an open, welcoming, diverse, inclusive, and healthy
community — assuming good faith, giving and accepting constructive feedback gracefully, and
focusing on what is best for the project.

Unacceptable behavior includes harassment, insulting or derogatory comments, personal or
political attacks, publishing others' private information, and other conduct reasonably
considered inappropriate in a professional setting.

Maintainers are responsible for clarifying and enforcing these standards and may remove or
reject contributions that do not align with this Code of Conduct.

Report concerns privately to the maintainers (see [SECURITY.md](SECURITY.md) for a private
channel). Reports are handled confidentially.

The full text is at https://www.contributor-covenant.org/version/2/1/code_of_conduct/.
